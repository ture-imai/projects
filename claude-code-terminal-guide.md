## ターミナル版 Claude Code での手順

**1. ターミナルを開き、対象フォルダに移動**
```bash
cd "C:\Users\SQR\OneDrive - 株式会社トゥルーステージ\【A】重要\R-C"
```

**2. Claude Code を起動**
```bash
claude
```

**3. あとは今回と同じ日本語でそのまま指示するだけ**
```
このフォルダのOとNの中に2025●月度のフォルダがあります。
このフォルダを、新たに2025というフォルダを作成し、その中に移動してください
```

---
## ポイント
- `cd` で**作業フォルダに移動してから起動**すると、Claude Code がそのフォルダを作業ディレクトリとして認識するので指示がシンプルになります
- 今回のブラウザ版（Claude Code with GUI）と**まったく同じ指示文**が使えます
- ターミナル版でも内部でbashコマンドを実行するので結果は同じです
